# 🧔 Jay Mata Di Salon - Premium Grooming Website

A modern, responsive website for a premium salon in Shamgarh, Madhya Pradesh. Built with HTML, Tailwind CSS, and vanilla JavaScript.

## ✨ Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Service Showcase** - Display all salon services with pricing
- **AI Style Advisor** - Personalized hair cutting recommendations based on face shape and hair type
- **WhatsApp Integration** - Direct booking through WhatsApp
- **Modern UI** - Dark theme with gold accents, smooth animations
- **Contact Information** - Easy access to salon details
- **SEO Optimized** - Meta tags and structured content

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript (Vanilla)** - Interactive features
- **Font Awesome** - Icon library
- **Google Fonts** - Premium typography (Cinzel & Montserrat)

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/vyasnageshwar599-sketch/jay-mata-di-salon.git
cd jay-mata-di-salon
```

### 2. Open in Browser
Simply open `index.html` in your web browser. No build process or dependencies required!

```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# Or double-click index.html in your file explorer
```

### 3. Deploy Online

#### Option 1: GitHub Pages (Free)
1. Push code to GitHub repository
2. Go to Settings → Pages
3. Select `main` branch as source
4. Your site will be live at `https://vyasnageshwar599-sketch.github.io/jay-mata-di-salon/`

#### Option 2: Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Click "New site from Git"
3. Connect your GitHub repository
4. Deploy automatically

#### Option 3: Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. Deploy with one click

## 🎨 Customization

### Change Colors
Edit the Tailwind config in `index.html`:
```javascript

        gold: '#FFD700',     // Primary color
        dark: '#0a0a0a',     // Background
        card: '#161616',     // Card background
        input: '#222222'     // Input background

}
```

## 🐛 Troubleshooting

### WhatsApp not opening
- Ensure WhatsApp is installed on your device
- Check that phone number is correct
- Try opening WhatsApp Web if on desktop

### Date/Time picker not working
- Use modern browser (Chrome, Firefox, Safari, Edge)
- Older browsers may need polyfills

### Styles not loading
- Clear browser cache (Ctrl+Shift+R or Cmd+Shift+R)
- Check internet connection for CDN resources
- Ensure all files are in the same directory
